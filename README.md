# 🎮 Color Ball Sort Puzzle

## 📖 Tentang Game

**Color Ball Sort Puzzle** adalah permainan puzzle berbasis desktop yang dikembangkan menggunakan **Python**, **Tkinter**, dan **MySQL**. Game ini menantang pemain untuk mengelompokkan bola-bola berwarna ke dalam tabung hingga setiap tabung hanya berisi satu warna yang sama.

## 🎯 Tujuan Permainan

Susun dan kelompokkan seluruh bola berdasarkan warnanya dengan memindahkan bola antar tabung sesuai aturan yang berlaku. Selesaikan setiap level sebelum waktu habis untuk mendapatkan skor tertinggi.

## 🕹️ Cara Bermain

1. Klik tabung sumber untuk memilih bola teratas.
2. Klik tabung tujuan untuk memindahkan bola.
3. Bola hanya dapat dipindahkan ke:

   * Tabung kosong.
   * Tabung dengan warna bola teratas yang sama.
4. Selesaikan puzzle dengan mengelompokkan seluruh bola berdasarkan warna.
